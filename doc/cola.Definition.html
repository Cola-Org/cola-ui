on(eventName,listener,once)绑定一个事件cola.Element
Parameters:
  • eventNamestring
    利用事件名称，我们可以为某个事件监听器定义别名。例如 click:temp 表示我们要为ready事件绑定事件，同时指定改事件监听器的别名为temp。

  • listenerfunction
    事件监听器方法
  • onceboolean
    实际使用时可以利用下面的方法简化这种使用方式 Element

Returns:cola.Element

button.on("click", function() { ... });


one(eventName,listener)绑定一个一次性事件cola.Element
Parameters:
  • eventNamestring
    事件名
  • listenerfunction
    事件监听器方法
Returns:cola.Element

button.one("click", function() { ... });


off(eventName)根据事件名解绑一个事件监听器cola.Element
Parameters:
  • eventNamestring
    如果我们不在事件名中定义别名，那么此方法会解绑该事件下的所有监听器。
如果我们在事件名中定义了别名，那么此方法只会解绑与该别名匹配的监听器。

Returns:cola.Element

button.off("click"); // 解绑ready事件下的所有监听器
button.off("click:temp"); // 解绑ready事件下别名为temp的监听器
